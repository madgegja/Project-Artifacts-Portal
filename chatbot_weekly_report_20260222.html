CORE INSIGHT — 실데이터 11,621건 분석
CS 상담의 83%가 반복적이지만 반드시 거쳐야 하는 정보 확인 단계
결제정보 요청 45% 구독취소 확인 18% 환불 처리 10% 환불 거절 7% 계정 삭제 3%

기존 CS 왕복 vs AI 스마트 레이어링

기존 — 반복 왕복→AI — 티켓 전 정보 확보
문의→정보요청→회신대기→확인→안내 (3~5회 왕복) → 티켓 전 AI가 양식 안내 + 정보 확인 완료
운영자가 매번 동일 정보 반복 요청 (83%) → 고객이 사전 제공 → 운영자 즉시 처리
정보 부족 → 추가 왕복 → 첫 응답 해결율 낮음 → 필요 정보 미리 확보 → 첫 응답 해결율 3~4배
야간/주말 응답 불가 → 24/7 AI 안내 → 언제든 양식 확인 + 티켓 생성
