  • ID 중복 사용 통계, VIP회차 부정접근 통계 활용

2.1 데일리 모니터링

Allaine(알레인)님이 수행하는 데일리 모니터링은 4가지 축으로 운영됩니다.

업무 1. 보안마크(Security Dot) 검사 — 불법 유출 계정 특정

투믹스 웹사이트/앱은 웹툰 페이지에 삼각형 보안마크를 유저별로 고유하게 랜덤 배치합니다. 불법 사이트에 유출된 이미지에서 이 마크를 찾으면 유포자를 특정할 수 있습니다.

검사 절차

  1. 불법 사이트에서 최신 챕터를 확인
  2. 페이지를 꼼꼼히 확인하여 삼각형 보안마크 발견
  3. 투믹스 플랫폼에서 동일 챕터 찾기 → 원본 챕터 URL 복사
  4. G관리자 > 부정 사용자 관리 > 보안 마크 검색 도구에 URL + 마크 위치 입력
  5. 불법 유포 회원 ID 확인 → 제재 처리 (섹션 3.3 참조)

업무 2. Google 도메인 저작권 검토 (Domain Review)

G관리자 도구가 키워드 검색으로 자동 수집한 의심 도메인을 일일 300~500건 검토합니다.

판정기준
Illegal투믹스 콘텐츠를 도용한 것이 확인된 도메인
Legal투믹스 콘텐츠 무관, 합법적인 도메인
Neutral불확실 (로딩 안됨, 1분 이상 로딩, 콘텐츠 없으나 비합법적)

절차

  1. G관리자 > 부정 사용자 관리 > 도메인 저작권 검토(Google)에서 "Google search" 클릭
  2. 관련 링크를 여러 개 클릭하여 도메인 확인
  3. Illegal / Legal / Neutral 판정
  4. 필요 시 판정 변경 가능 (리뷰 리스트에서)
업무 3. KR 불법 유통 링크 확인
  • 국내 불법 유통 사이트에서 투믹스 작품이 유통되는 링크를 탐지
  • 탐지된 링크는 박선주/김지호님께 공유
업무 4. 수동 DMCA 신고
  • 수집된 불법 URL을 Google DMCA 보고서로 제출 (상세 절차는 섹션 2.2)
  • 월간 신고 건수: 17,800 ~ 52,000건 (월별 변동)
  • Allaine은 신고(Report) 업무만 담당, 탐지/제거는 MARQ·COA가 처리
