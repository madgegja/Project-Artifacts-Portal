Phase 1 — Quick Wins

셀프서비스 전환으로 38% 티켓 즉시 감소
Phase 1
🔄
1-1. 구독 해지 셀프서비스 UI 구축
앱/웹 내 1-Click 구독 해지 버튼 도입. 현재 "Please cancel my subscription" 수동 요청 → 즉시 자동 처리. EN/FR/ES/PT/DE 5개 언어 지원 필수.
~400건/월 감소 전체 24% 해소 완전 자동화
🔐
1-2. 비밀번호 재설정 자동화
Zendesk 매크로 + 자동 발송 파이프라인. 비밀번호 리셋 키워드 감지 → 자동 리셋 이메일 트리거. 기존 수동 링크 안내 → 완전 자동 대응.
~150건/월 감소 전체 9% 해소 매크로 자동화
🗑️
1-3. 계정 삭제 요청 프로세스 간소화
본인인증 포함 계정 삭제 요청 폼 구축. 3단계 수동 프로세스(본인확인→삭제→통보) → 1-Step 자동 파이프라인.
~80건/월 감소 전체 5% 해소
